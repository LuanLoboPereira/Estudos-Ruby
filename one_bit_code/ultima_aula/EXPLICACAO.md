# Leitura de arquivos

-> Ler e escrever em arquivos te dá o poder de criar uma grande gama de **solftwares**, felizmente fazer isto no Ruby é bem fácil

# Chamadas WEB

-> Através de chamadas Web podemos **conectar o nosso solftware** a outros programas através da internet. Então veremos alguns exemplos de como fazer isso com a Lib **Ner::HTTP** do Ruby.

# Web Scraping

-> Através do Web Scraping conseguimos **extrair dados** de sites, como por exemplo:

- Resultados dos jogos de futebol de um portal de notícias
- Uma lista de post em um blog
- Valor de dolár ou BitCoin
- E _muito_ mais...
